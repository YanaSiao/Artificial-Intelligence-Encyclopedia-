# Artificial-Intelligence-Encyclopedia
Personal Encyclopedia about foundational AI concepts, practical application, and modern trends. The notes are mainly based on my **Master of Computer Science and Engineering** courses (Politecnico di Milano).

### The Goal
The goal of this project is to create a coherent, intertwined network of knowledge around my area of expertise. Rather than isolated files, this vault uses **Maps of Content (MOCs)** to link theory, math, and practical code.

### How to Use
These notes are written in **Markdown** and optimized for **[Obsidian](https://obsidian.md/)**.

- To see the "intertwined network," I recommend opening this folder in Obsidian and checking the **Graph View**.
- Start your journey at the `00_MOCs`.
